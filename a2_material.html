System Vulnerabilities

Understanding System Vulnerabilities

  • Network Vulnerabilities
    • Firewall Ports: Open or improperly configured firewall ports can be exploited by attackers.
    • External Storage Devices: External devices like USB drives can introduce malware if not properly managed.
  • Organisational Vulnerabilities
    • File Permissions or Privileges: Incorrect permissions can allow unauthorized access to sensitive data.
    • Password Policy: Weak or outdated password policies can be easily breached.
  • Software Vulnerabilities
    • Untrustworthy Sources: Software from unreliable sources can be malicious.
    • Downloaded Software: May contain malware or be vulnerable to exploitation.
    • Illegal Copies: Often lack necessary security patches.
    • SQL Injection: A code injection technique that can destroy your database.
    • Zero-Day Exploits: New vulnerabilities that are exploited before a patch is available.
  • Operating System Vulnerabilities
    • Unsupported Versions: Lack security updates making them vulnerable.
    • Updates Not Installed: Systems missing critical updates are exposed to known exploits.
  • Mobile Device Vulnerabilities
    • OEM Updates: Reliant on manufacturers for timely security updates.
  • Physical Vulnerabilities
    • Theft of Equipment: Loss of physical devices containing sensitive data.
    • USB Storage Devices: Risk of sensitive data being copied and stolen.
    • Social Engineering: Collecting passwords and other sensitive information through manipulation.
  • Process Vulnerabilities
    • System Use: Poor practices in how people use the system, including leaks and sharing security details.
  • Cloud Computing and IoT Vulnerabilities
    • Cloud Computing: Security implications of storing data in the cloud.
    • IoT Devices: Risks associated with Internet of Things devices, including lack of security updates.
  • Attack Vectors
    • Wi-Fi: Vulnerabilities in wireless network connections.
    • Bluetooth®: Risks of unauthorized access through Bluetooth connections.
    • Internet Connection: Threats from internet exposure and unsecured connections.
    • Internal Network Access: Internal threats from unauthorized access within the network.
