Network Types and Features

Understanding Network Types and Features

  • Applications and Features of Networks
    • Local Area Network (LAN): Connects devices within a small geographical area like an office.
    • Wireless LAN (WLAN): A LAN that uses wireless communication methods.
    • Wide Area Network (WAN): Connects devices over large geographical areas.
    • Storage Area Network (SAN): Provides access to consolidated block-level storage.
    • Personal Area Network (PAN): Connects devices within the range of an individual person.
    • Intranet: A private network accessible only to an organizationâ€™s staff.
    • Extranet: An intranet that is partially accessible to authorized outsiders.
    • Internet: A global network connecting millions of private, public, academic, business, and government networks.
    • Cloud: Delivery of computing services over the internet.
    • Wired and Wireless Integration: Combining wired and wireless network technologies.
  • Network Topologies
    • Physical Topologies:
      • Star: All nodes are connected to a central hub.
      • Extended Star: A star network with additional hubs.
      • Hierarchical: A star network with multiple levels.
      • Wireless Mesh: Nodes connect dynamically and directly.
      • Ad-Hoc: A mix of wired and wireless for BYOD environments.
    • Logical Topologies:
      • Ethernet Standards: Standards for wired (IEEE 802.3) and wireless (IEEE 802.11) networking.
  • Network Architecture
    • Peer to Peer: All devices share resources equally.
    • Client/Server: Clients request resources and services from a central server.
    • Thin Client: Computing model where client devices rely on server resources.
  • Modern Trends
    • Virtualisation: Creating virtual versions of physical devices and resources.
    • Cloud Computing: Delivery of computing services over the internet.
    • BYOD: Allowing employees to use their personal devices for work.
    • Software-Defined Networking (SDN): Network management approach that enables dynamic, programmatically efficient network configuration.
    • Storage-Defined Networks: Managing storage resources and data traffic through software.
    • Internet of Things (IoT): Network of physical objects embedded with sensors and connectivity to exchange data.
  • Network Schematic Diagrams
    • Interpretation: Ability to read and understand network diagrams.
    • Amendment: Making changes to network diagrams using suitable software.
