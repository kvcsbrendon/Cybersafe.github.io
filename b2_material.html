Network Components

Understanding Network Components

  • Hardware Components
    • End-User Devices:
      • Mobile Devices: Smartphones, tablets, and laptops used by end users.
    • Connectivity Devices:
      • Switches: Devices that connect and manage data traffic between multiple devices on a network.
      • Routers: Devices that forward data packets between computer networks.
      • Access Points: Devices that allow wireless devices to connect to a wired network.
      • Multi-Functional Devices: Devices that combine several networking functions, such as routers with built-in switches and access points.
      • USB Hubs: Devices that expand a single USB port into several for connecting multiple peripherals.
      • Modems: Devices that modulate and demodulate signals for data transmission over phone lines or cable.
    • Connection Media:
      • Cable: Physical cables like Ethernet for wired connections.
      • Wireless: Technologies such as Wi-Fi, Bluetooth, and infrared (IR) for wireless connections.
      • Fibre: Fibre optic cables for high-speed data transmission.
      • Li-Fi: Light-based data transmission technology.
  • External Media and Storage
    • Flash Drives: Portable storage devices for data transfer and storage.
    • Optical Media: Storage media like CDs, DVDs, and Blu-ray discs used for data storage and distribution.
  • Software Components
    • Network and Device Operating Systems: Operating systems that manage network and device functions.
    • Network Monitoring, Management and Troubleshooting Tools:
      • Performance Monitor: Tool to monitor and analyze network performance.
      • Events and Logs Viewer: Tool to view and analyze system events and logs.
      • Vulnerability Scanners: Tools to identify security vulnerabilities in networks and systems.
      • Packet Sniffers: Tools to capture and analyze network traffic.
    • Network Applications:
      • Database Tools: Applications for managing and accessing databases.
      • Document Management Tools: Applications for managing and sharing documents.
      • Network Discovery Tools: Applications for identifying and mapping network devices and services.
